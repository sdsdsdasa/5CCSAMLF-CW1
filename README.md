# 5CCSAMLF-CW1 Machine Learning Data Challenge

This project builds a regression model to predict the continuous target variable **`outcome`** using the provided training dataset and generates predictions for the unseen test dataset.

The workflow follows the coursework pipeline:

1. Exploratory Data Analysis (EDA)  
2. Preprocessing  
3. Model Selection  
4. Model Training and Hyperparameter Tuning
5. Final Prediction (Submission File)

## How to Run Each Script (Steps)

### 1. Exploratory Data Analysis (EDA)

Open and run the notebook:

``` bash
jupyter notebook notebooks/EDA.ipynb
```

This step: - Analyses data distributions - Checks correlations and feature importance - Detects missing values and outliers - Generates plots in outputs/figures

------------------------------------------------------------------------

### 2. Baseline Model (Optional)

Run the simple linear regression baseline:

``` bash
python src/baseline_eval.py
```

This script: - Applies one-hot encoding - Trains a Linear Regression model - Generates a valid submission file - Check avaliability of the given script

------------------------------------------------------------------------

### 3. Model Selection

Compare multiple models:

``` bash
python src/train_models.py
```

This script: - Trains several regression models (Linear, Ridge, Lasso, ElasticNet, RandomForest, GradientBoosting) - Evaluates using validation R² - Prints model ranking - Saves predictions using the best model

------------------------------------------------------------------------

### 4. Model Training and Hyperparameter Tuning (Final Model)

Run:

``` bash
python src/tune_model.py
```

This script: - Tunes Random Forest and Gradient Boosting hyperparameters - Uses validation R² to select the best model - Retrains on the full training dataset - Generates the final submission file

Output:

    outputs/CW1_submission_K23158987.csv

------------------------------------------------------------------------

### 5. Submission File Requirements

The submission file must:

-   Contain **one column only**
-   Column name must be **`yhat`**
-   Have the same number of rows as `CW1_test.csv`
-   Contain **no missing values**

This file is automatically generated by:

    train_models.py
    or
    tune_model.py

------------------------------------------------------------------------

## Reproducibility

-   All scripts use a fixed random seed (`np.random.seed(123)`)
-   Training and test features are aligned to avoid column mismatch
-   No data leakage occurs (training only uses training data)
